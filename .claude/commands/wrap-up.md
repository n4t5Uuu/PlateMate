# /wrap-up

Update `.claude/HANDOFF.md` to reflect the current state of the session so the next session can pick up exactly where this one left off.

## Steps

1. Review what was discussed and built in this session
2. Update the "What's Been Built" section with anything new
3. Update the "What's Next" checklist — check off completed items, add new ones
4. Update the "Known Issues" section with any new findings
5. Update the "Last updated" date at the top

## Format

Keep it factual and short. Focus on:
- What changed (not what was discussed)
- What's blocked or broken
- What the next session should start with
