# Behavior Rules

## Backend Modifications
- **Display Code Only**: When a task is related to the backend (such as Supabase database helpers, SQL files, migrations, database functions, schema definitions, or API endpoints), do NOT modify or write files directly to the filesystem. 
- **Exception**: Only display the proposed code blocks/diffs in the chat response for the user to review and handle, unless the user explicitly instructs you in that specific turn to apply/write it for them.
